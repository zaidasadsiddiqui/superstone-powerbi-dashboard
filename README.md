Superstore Sales Performance Dashboard
Project Overview
This project analyzes the Superstore dataset to evaluate sales performance, profitability, and customer segmentation using Power BI.

Tools Used
Power BI
DAX (for KPI calculations)
Excel (data source)

KPIs Included

Total Sales
Total Profit
Total Orders
Profit Margin (%)

Key Insights

Northwest Territories generated the highest sales.
Technology category contributed strong revenue.
Overall profit margin is approximately 9%.
Sales peaked around 2010 and declined afterward.

Dashboard Features

Interactive slicers (Region and Year)
Sales & Profit comparison
Category and segment performance analysis
Yearly sales trend visualization
